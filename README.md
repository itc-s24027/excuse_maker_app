# 言い訳生成アプリ
制作日： 2年生 12月
制作期間： 2ヶ月

⚠まだ制作途中のプロジェクトです

### 概要
学校イベントでの学生出席確認を目的に、楽しめるスタンプラリーアプリをチームで制作しました。
受付のQRコードからGoogleログインを行い、データベース登録名で出席を確認。各階のパスワードを使った6問の問題に挑戦し、全問正解者には景品を配布。さらにシークレット問題も用意し、追加景品を提供しました。

### スクリーンショット

### 機能
* 新規チャット作成
* 生成された言い訳が成功したかの判定
* 成功した言い訳を共有する機能
* 共有された言い訳をいいねする機能
* ランキング表示機能
